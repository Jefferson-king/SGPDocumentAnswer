题目：给定一个含有 n 个正整数的数组和一个正整数 s ，找出该数组中满足其和 ≥ s 的长度最小的 连续 子数组，并返回其长度。如果不存在符合条件的子数组，返回 0。

- 思路：滑动窗口法：先找到总和大于S的一个数组和sum，假设起点为start,终点为end，一旦大于sum了，起点往右移动一个单位，和减去移动前的num[start]用while循环判断直到小于sum,然后终点往右移动一个单位，判断是否已经到nums.length,往复循环

```
/**
 * @param {number} target
 * @param {number[]} nums
 * @return {number}
 */
var minSubArrayLen = function(target, nums) {
    let end=0,n=nums.length,start=0
    let sum=0,ans=Infinity
    while(end<n){
        sum +=nums[end]
        while(sum >= target){
            ans = Math.min(ans,end-start+1)
            sum -=nums[start++]
        }
        end++;
    }
    return ans === Infinity ? 0:ans
   
};
```
