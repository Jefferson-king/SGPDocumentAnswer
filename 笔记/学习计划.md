第 3 周：RAG 与 Pinecone

Day 15｜2026-04-09｜5h

- 学 RAG 全流程：切片、embedding、召回、拼接上下文、回答
- 理解 chunk size、overlap、top-k、metadata
- 写一页“RAG 为什么有效”的总结
验收：你能画出 RAG 数据流
参考文档：[rag-learning-plan.md](./rag-learning-plan.md)

Day 16｜2026-04-10｜5h

- 选文档解析方案，支持 TXT、PDF
- 实现文本提取与分块
- 设计 chunk 规则：长度、重叠、页码、来源

Day 17｜2026-04-11｜5h

- 注册并配置 chroma
- 创建 index，测试 upsert 和 query
- 理解相似度、距离、top-k
验收：可以根据一句查询搜回相关 chunk

Day 18｜2026-04-12｜6h

- 写 ingestion 流程：上传文件 -> 解析 -> 切片 -> embedding -> 入库
- 给每个 chunk 记录 source、page、chunkId
- 运行一次完整入库流程
验收：1 份真实文档能完整入库

Day 19｜2026-04-13｜5h

- 实现 retrieval API
- 查询时先检索 top-k，再把片段拼入 prompt
- 返回答案时保留引用信息
验收：回答里能带来源片段

Day 20｜2026-04-14｜5h

- 准备 15~20 个测试问题
- 记录错误类型：答非所问、幻觉、引用错、检索不到
- 总结 chunk 质量问题
验收：形成第一版评测表

Day 21｜2026-04-15｜4h

- 优化 chunk size、overlap、top-k、阈值
- 测试“无答案时拒答”逻辑
- 输出一份 RAG 调参结论
验收：回答质量比 Day 19 明显提升
